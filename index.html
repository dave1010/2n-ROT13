2n-ROT13 encoder / decoder

About 2n-ROT13

ROT13 is an awesome algorithm. It's a special version of the Caesar cipher, that can be enhanced by using multiple rounds. It's well battle-tested as it's been around for many years.

Security experts currently recommend using bcrypt, scrypt or PBKDF2 for storing passwords. These methiods use multiple rounds of simple algorithms such as SHA1 and Blowfish. The algorithms themselves are cracked very simply but using multiple rounds makes brute-forcing much harder.

In the same way, ROT13 can be applied multiple times to make it harder to brute force. Here's an important looking scientific paper (it's a PDF, so it must be reliable), explaining the benefits of applying just 2 rounds of ROT13 (2-ROT13). One of these benefits is that 2-ROT13 has never been brute-forced.

My old favourite encryption method was bcrypt, which applies 2n rounds to the Blowfish cipher. I've taken the best bits of bcrypt (the 2n rounds) and the best bits of 2-ROT13 (the strength against brute-force) to create 2n-ROT13.
